# Caderno Temático com NotebookLM

## Contexto e Objetivos

Este repositório foi desenvolvido como parte de um desafio prático com foco no uso da Inteligência Artificial como ferramenta de aprendizagem ativa.

O objetivo é construir um caderno temático utilizando o NotebookLM, organizando informações relevantes a partir de fontes confiáveis e aplicando técnicas de engenharia de prompts para aprofundar o entendimento sobre o tema escolhido.

Tema do estudo: Introdução à Análise de Dados

Objetivos principais:
- Entender possiveis areas;
- O que precisamos saber;
- Desenvolver habilidades de interação com IA;

## Engenharia de Prompts e Análises

Durante o desenvolvimento, foram utilizados diferentes prompts para explorar o conteúdo das fontes.

### Exemplos de Prompts Utilizados

- "Quais são as linguagens de programação mais essenciais para análise?"
- "Resuma os principais pontos deste material"
- "Quais são as diferenças entre os perfis de analista e cientista de dados?"
- "Quais ferramentas de Business Intelligence são recomendadas além do SQL?"

### Ajustes e Iterações

Algumas estratégias utilizadas para melhorar os resultados:

- Refinamento das perguntas para torná-las mais específicas
- Solicitação de exemplos práticos
- Comparação entre diferentes respostas geradas

### Dificuldades Encontradas

- Respostas muito genéricas em prompts amplos
- Necessidade de detalhamento maior em alguns conceitos
- Ajuste na forma de perguntar para obter respostas mais completas

## Miniguia de Estudo

### Resumo do Tema

A área de dados dedica-se a transformar informações brutas em insights estratégicos para impulsionar a tomada de decisão nas organizações
. Ela abrange diversas especializações, sendo as principais o Cientista de Dados (focado em modelos preditivos e IA), o Analista de Dados (focado em interpretar tendências atuais) e o Engenheiro de Dados (responsável pela infraestrutura e coleta)
.
Para atuar nesse setor, as habilidades fundamentais incluem:
Linguagens de Programação: Especialmente SQL para bancos de dados e Python ou R para análise estatística
.
Ferramentas de BI: Domínio de planilhas (Excel/Google Sheets) e softwares de visualização como Power BI ou Tableau
.
Conhecimentos Base: Sólida noção de estatística, matemática e visão de negócios
.
O mercado para esses profissionais apresenta alta demanda, segurança no emprego e salários atrativos, com previsões de crescimento significativas para os próximos anos
