# Repositório de Estudos em Dados

Este repositório foi criado com o objetivo de armazenar e organizar atividades desenvolvidas durante um curso voltado para a área de dados.

As atividades são realizadas utilizando notebooks, conforme proposto pelo curso, e incluem exercícios práticos, exemplos e aplicações dos conceitos estudados.

## Conteúdo

O repositório pode conter:

- Exercícios em SQL  
- Análises de dados em notebooks  
- Práticas com ferramentas da área de dados  
- Exploração de conceitos fundamentais  

## Observação

Este repositório tem caráter acadêmico e faz parte do processo de aprendizagem. O conteúdo pode ser atualizado conforme o avanço do curso.
